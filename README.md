# Как установить и использовать???

Очень просто! Загружаем customtroll.deb и устанавливаем в терминале коммандой:

    sudo dpkg --install customtroll.deb

Далее зависимости:

    sudo apt install -f

И запускаем:

    sudo customtroll rickroll
